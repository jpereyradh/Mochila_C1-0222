Git Bash es la terminal de git que se puede utilizar desde los dispositivos que contengan un sistema operativo diferente a Linux. Es una CLI, esto es una interfaz de línea de comandos. En cambio Visual Studio Code es un IDE (Entorno de Desarrollo Integrado), el cual contiene un conjunto de herramientas que facilitan la creación y desarrollo de software, que a su vez permite vincular la terminal de Git Bash.
