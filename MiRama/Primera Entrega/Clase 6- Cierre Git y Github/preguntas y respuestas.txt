a. ¿Quién inventó el sistema de control de versión Git y por qué?

El sistema de control de versiones de Git fue inventado por Linus Torvalds, el creador de Linux.

b. ¿A quién pertenece actualmente Github y por qué?

Microsoft es actualmente el dueño de GitHub, lo adquirió teniendo como objetivo aumentar
 el uso empresarial de GitHub y llevar las herramientas y servicios de desarrollo de Microsoft a nuevas audiencias.
