a. ¿Quién inventó el sistema de control de versión Git y por qué?
    Git es un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

b. ¿A quién pertenece actualmente Github y por qué?
    Actualmente pertenece a Microsoft, quien planea que la comunidad de desarollo de software sea mas libre y abierta, "reforzando el compromiso con la libertad, la apertura e innovacion" dijo Satya Nadella, consejero delegado de Microsoft.
