quien invento el sistema de control de versiones git  y porque?

Git es un proyecto de código abierto maduro y con un mantenimiento activo que desarrolló originalmente Linus 
Torvalds, el famoso creador del kernel del sistema operativo Linux, en 2005, 
pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen 
un gran número de archivos de código fuente.

¿a quien pertenece actualmente github y porque?
Actualmente le pertenece a microsoft y el porque esta basado en la competencia de mercado y la intencion de la compania de poder posicionar a su proyecto, gitlab como el maximo competidor


¿hay otra forma que no sea la terminal para trabajar en github?

En github uno puede gestionar el repositorio remoto sin uso de terminal, pero si uno quiere gestionarlo de manera local debera hacer uso de la terminal, ya sea la de git o la de algun 
editor de codigo.
